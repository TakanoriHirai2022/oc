教育プログラム

文部科学省認定・リテラシーレベル

プログラムの概要

数理・データサイエンス・AIへの関心を高め、これからの社会で必須となる基礎スキルを網羅した体系的プログラムです。

💡 全学部の学生が受講可能！

スキルを武器にしたい20代のキャリアを強力にバックアップします。

データサイエンス基礎 Ⅰ

1年次前期開講 ／ 2026年度より全学科必修

基礎知識

AIやネットワークの仕組み、情報社会を生きるためのモラルを学びます。

情報活用

PC操作はもちろん、デジタル社会の基盤となる情報リテラシーを固めます。

発信技術

データを的確に伝えるためのスライド作成・プレゼン技法を習得します。

データサイエンス基礎 Ⅱ

1年次後期開講 ／ 日常生活やビジネスで使いこなす

データを分析し、未来を予測・判断するための実践的なアプローチを体験します。

  • 可視化： グラフやヒートマップによるチャート化
  • データ比較： 条件を揃えた比較やA/Bテストの基本
  • 生成AI理解： アルゴリズムの技術的な仕組みの学習

Python と SQL の修得

本格的なスキルを、初心者向けのステップで基礎から実習します。

  • Python (pandas)： データのクレンジングや前処理
  • SQL： データの整理と管理システムの理解
  • データ収集： スクレイピングやWebAPIの活用

プログラムで身につく成果

可視化能力

データを綺麗にまとめ、誰にでも伝わる図表にする力。

AI適切な活用力

ただ使うだけでなく、仕組みを理解してAIを業務に応用する力。

データアクセス

SQLを使って、社内のデータに素早く安全にアクセスする環境を築く力。

修了要件と国の認定

【修了要件】 2科目4単位の修得

「データサイエンス基礎 Ⅰ」（2単位） ＋ 「データサイエンス基礎 Ⅱ」（2単位）

【継続学習】 2科目4単位

「データ分析演習」（2単位） ＋ 「プログラミング演習」（2単位）
